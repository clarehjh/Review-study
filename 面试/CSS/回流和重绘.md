# 回流 reflow

js 操作影响节点大小，位置修改重新计算

# 什么情况引起回流

dom 结构发生改变（添加新的节点或移除节点）
改变了布局（修改了 width,height,padding,font-size 等值）
比如窗口 resize(修改了窗口的尺寸等)
比如调用 getComputedStyle 方法获取尺寸，位置信息

# 什么是重绘

第一次渲染的内容称为绘制
之后重新渲染称为重绘

# 什么情况下引起重绘

比如修改背景颜色，文字颜色，边框颜色，样式等
